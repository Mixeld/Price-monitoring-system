# Price-monitoring-system
Проект написанный в рамках выполнения работ по ПнаЯВ (Java)


======= ОПИСАНИЕ ПРОЕКТА =======

Система позволяет собирать данные о ценах по расписанию, отслеживать историю изменений и получать уведомления при снижении цены до заданного порога.

======= ОСНОВНЫЕ ОСОБЕННОСТИ =======

Парсинг цен: Сбор актуальных данных о товарах по URL.
История изменений: Хранение всей динамики цен в базе данных для последующего анализа.
Планировщик задач: Гибкая настройка периодичности проверки цен (каждый час, раз в сутки и т.д.).
Алертинг: Отправка уведомлений (Email/Telegram) при достижении целевой цены.
REST API: Управление списком отслеживаемых товаров и просмотр статистики.
Визуализация: Простой UI для отображения графиков изменения цен (опционально).

======= ТЕХНИЧЕСКИЙ СТЕК =======
*   **Backend:**
    *   Java 17
*   **База данных:**
    *   PostgreSQL (основная БД)
